Instalar GhostAssist

Overlay invisível para entrevistas técnicas

1

Qual é o seu Mac?

Clique na maçã () → Sobre este Mac → veja o chip.

⬇ Baixar GhostAssist
2

Instale o app

Abra o .dmg baixado e arraste o GhostAssist para a pasta Applications.

3

Libere o app no macOS

Abra o Terminal e rode o comando abaixo — isso é necessário uma única vez:

xattr -cr /Applications/GhostAssist.app clique para copiar ✓ copiado
⚠️ O macOS bloqueia apps sem assinatura Apple com a mensagem "está danificado". Este comando remove o bloqueio — é seguro.
4

Abra o GhostAssist

Clique duas vezes em GhostAssist.app em Applications. O app aparece como um overlay invisível — sem ícone no Dock.

GhostAssist avisa automaticamente quando há uma nova versão disponível.
